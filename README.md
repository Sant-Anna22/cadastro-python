# Sistema de Cadastro em Python

Projeto simples de cadastro de usuários feito em Python no terminal.

## Funcionalidades

- Cadastrar usuários
- Listar usuários
- Apagar usuários
- Validação de dados
- Confirmação antes de apagar cadastro

## Conceitos praticados

- Funções
- Listas
- Dicionários
- Loops
- Condicionais
- try/except
- enumerate()
- Manipulação de strings

## Tecnologias

- Python 3

## Objetivo

Projeto desenvolvido para praticar lógica de programação e estruturação de sistemas simples em Python.
